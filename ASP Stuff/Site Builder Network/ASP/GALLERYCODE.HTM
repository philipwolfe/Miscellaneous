    To call the viewpics.asp fill each url of the thumbnail with the appropriate variables.

    IMG   - the location of the larger image.
    HEIGHT- height of the image
    WIDTH - width of the image
    INFO  - information displayed about the image

    <A HREF="viewpics.asp?TITLE=Heather+in+a+window&IMG=/sitebuilder/columnists/images/hands.jpg&HEIGHT=200&WIDTH=141&INFO=Black+and+white+photograph+of+my+friend+Heather+in+Paul's+window.">

    </a>
