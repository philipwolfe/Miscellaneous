These files customize the VS.NET Start page to show you the language of the projects in the MRU.

To use it:
1. Browse to this folder: C:\Program Files\Microsoft Visual Studio .NET\Common7\IDE\HTML\1033
2. Make backup copies of the files you are about to replace.
3. Copy the files contained in this zip file to this folder.
4. If VS.NET is open, refresh the start page.
